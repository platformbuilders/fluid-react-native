# Introdução

O Fluid React Native é uma biblioteca de componentes React Native que implementa o Design System Fluid da Platform Builders.

## O que é o Fluid React Native?

Fluid React Native oferece uma coleção abrangente de componentes reutilizáveis para desenvolvimento mobile com React Native, seguindo as diretrizes do Design System Fluid. Esses componentes são projetados para ajudar os desenvolvedores a criar interfaces consistentes, acessíveis e de alta qualidade.

## Principais características

- **Componentização**: componentes reutilizáveis e encapsulados
- **Acessibilidade**: suporte integrado para recursos de acessibilidade
- **Personalização**: temas e estilos personalizáveis
- **Testes**: alta cobertura de testes para garantir qualidade
- **Documentação**: documentação completa e exemplos de uso
