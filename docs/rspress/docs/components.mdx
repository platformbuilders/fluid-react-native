# Componentes

O Fluid React Native oferece uma ampla gama de componentes UI prontos para uso, implementando o Design System Fluid.

## Componentes Básicos

Componentes fundamentais para criar interfaces:

- [Typography](/components/typography) - Componente para texto com diferentes variantes
- [Touchable](/components/touchable) - Área clicável com feedback tátil
- [Link](/components/link) - Links e âncoras para navegação
- [Button](/components/button) - Botões em diferentes variantes
- [Icon](/components/icon) - Ícones para interfaces
- [Avatar](/components/avatar) - Representação visual de usuários
- [Image](/components/image) - Imagens com carregamento progressivo
- [Badge](/components/badge) - Indicadores numéricos e de status
- [Accordion](/components/accordion) - Seções de conteúdo retráteis

## Componentes de Navegação

Componentes para criar experiências de navegação:

- [BottomNavigation](/components/bottomnavigation) - Navegação inferior para aplicativos móveis

## Componentes de Formulário

Componentes para criação de formulários:

- [TextInput](/components/textinput) - Campo de entrada de texto
- [DatePicker](/components/datepicker) - Seletor de data e hora
- [PinInput](/components/pininput) - Input para códigos de verificação
- [SearchInput](/components/searchinput) - Input otimizado para busca
- [FormError](/components/formerror) - Exibição de erros em formulários
- [Checkbox](/components/checkbox) - Caixas de seleção
- [RadioButton](/components/radiobutton) - Botões de opção única
- [Toggle](/components/toggle) - Alternador de estados

## Componentes de Exibição

Componentes para exibir conteúdo:

- [UploadPhoto](/components/uploadphoto) - Upload e visualização de fotos
- [Markdown](/components/markdown) - Renderização de conteúdo markdown

## Componentes de Feedback

Componentes para feedback ao usuário:

- [Alert](/components/alert) - Alertas informativos
- [Modal](/components/modal) - Janelas modais
- [Spinner](/components/spinner) - Indicadores de carregamento
- [Placeholder](/components/placeholder) - Esqueletos de carregamento

## Categorias de Componentes

Os componentes do Fluid React Native são organizados nas seguintes categorias:

- **Básicos**: Componentes fundamentais como ícones e tipografia
- **Navegação**: Componentes para navegação entre telas
- **Formulários**: Componentes para entrada e manipulação de dados
- **Feedback**: Componentes para fornecer feedback ao usuário
- **Exibição**: Componentes para exibir conteúdo e informações

## Como usar

Todos os componentes podem ser importados diretamente do pacote principal:

```tsx
import { Button, Typography, Icon } from '@platformbuilders/fluid-react-native';

// Uso do componente
<Button title="Clique Aqui" onPress={() => {}} />;
```

## Personalizando Componentes

Todos os componentes suportam personalização através do sistema de temas e props específicas. Consulte a documentação de cada componente para detalhes sobre as opções de personalização disponíveis.
