# Button

O componente Button fornece um botão acessível para ações do usuário.

## Importação

```jsx
import { Button } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```jsx
<Button 
  onPress={() => console.log('Botão pressionado')}
  accessibility="botao-salvar"
>
  Salvar
</Button>
```

## Variações

### Variantes de Estilo

```jsx
// Preenchido (padrão)
<Button 
  onPress={handlePress}
  accessibility="botao-primario"
  variant="filled"
  colorVariant="primary"
>
  Botão Primário
</Button>

// Delineado
<Button 
  onPress={handlePress}
  accessibility="botao-secundario"
  variant="outline"
  colorVariant="secondary"
>
  Botão Secundário
</Button>

// Texto plano
<Button 
  onPress={handlePress}
  accessibility="botao-texto"
  variant="text"
>
  Botão Texto
</Button>
```

### Estados

```jsx
// Desabilitado
<Button 
  onPress={handlePress}
  accessibility="botao-desabilitado"
  disabled
>
  Botão Desabilitado
</Button>

// Carregando
<Button 
  onPress={handlePress}
  accessibility="botao-carregando"
  loading
>
  Salvando...
</Button>
```

### Com Ícones

```jsx
// Ícone à esquerda
<Button 
  onPress={handlePress}
  accessibility="botao-adicionar"
  leftIconName="plus"
>
  Adicionar
</Button>

// Ícone à direita
<Button 
  onPress={handlePress}
  accessibility="botao-proximo"
  rightIconName="arrow-right"
>
  Próximo
</Button>
```

## Acessibilidade

O componente Button implementa as seguintes propriedades de acessibilidade:

- `accessibilityLabel`: Descrição do botão para leitores de tela
- `accessibilityRole`: Definido como "button"
- `accessibilityState`: Informa os estados "disabled" e "busy" (quando loading=true)
- `accessibilityHint`: Informa que o botão ativa ao ser pressionado

### Exemplo com Acessibilidade Personalizada

```jsx
<Button 
  onPress={handlePress}
  accessibility="botao-concluir"
  accessibilityLabel="Finalizar pedido"
  accessibilityHint="Pressione para concluir seu pedido"
>
  Finalizar
</Button>
```

### Boas Práticas de Acessibilidade

1. **Labels descritivos**:
   - Use textos claros e específicos
   - Evite termos genéricos como "Clique aqui" ou "OK"

2. **Estados visíveis**:
   - Garanta que o estado desabilitado seja visualmente claro
   - Forneça feedback visual durante o carregamento

3. **Área de toque adequada**:
   - O botão tem área de toque mínima de 44x44dp
   - Considere botões maiores para alvos mais fáceis de tocar

## Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `id` | string | - | ID do componente |
| `accessibility` | string | Obrigatório | ID para acessibilidade |
| `accessibilityLabel` | string | Texto do botão | Descrição do botão para leitores de tela |
| `testID` | string | - | ID para testes |
| `children` | ReactNode | - | Conteúdo do botão |
| `onPress` | function | - | Função chamada ao pressionar o botão |
| `disabled` | boolean | false | Se o botão está desabilitado |
| `loading` | boolean | false | Se o botão está em estado de carregamento |
| `variant` | 'filled' \| 'outline' \| 'text' | 'filled' | Variante de estilo do botão |
| `colorVariant` | 'primary' \| 'secondary' \| 'success' \| 'danger' \| 'warning' \| 'info' | 'primary' | Variante de cor |
| `typographyVariant` | 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' | 'md' | Tamanho da tipografia |
| `leftIconName` | string | - | Nome do ícone à esquerda |
| `rightIconName` | string | - | Nome do ícone à direita |
| `contrast` | boolean | false | Aumentar contraste (para fundos escuros) |
| `rounded` | boolean | false | Se o botão deve ter cantos arredondados |
| `flat` | boolean | false | Se o botão deve ser plano (sem sombra) |
| `minWidth` | number \| string | - | Largura mínima do botão |
| `maxWidth` | number \| string | - | Largura máxima do botão |
| `style` | ViewStyle | - | Estilo customizado do botão |
| `textStyle` | TextStyle | - | Estilo customizado do texto |
