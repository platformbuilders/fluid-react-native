# Button

O componente `Button` é utilizado para iniciar ações e interações do usuário. Ele implementa várias variantes de estilo e cor para se adequar a diferentes contextos de uso.

## Importação

```tsx
import { Button } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
<Button
  title="Botão Primário"
  onPress={() => console.log('Botão pressionado')}
/>
```

## Variantes

O componente Button suporta diferentes variantes visuais:

```tsx
// Botão preenchido (padrão)
<Button children="Botão Preenchido" variant="filled" colorVariant="primary" />

// Botão com contorno
<Button children="Botão Outline" variant="outline" colorVariant="primary" />

// Botão fantasma (apenas texto)
<Button children="Botão Fantasma" variant="ghost" colorVariant="primary" />

// Botão com fundo suave
<Button children="Botão Tint" variant="tint" colorVariant="primary" />

// Botão plano (sem elevação)
<Button children="Botão Flat" variant="flat" colorVariant="primary" />
```

## Variantes de Cores

O Button suporta diferentes variantes de cor:

```tsx
<Button children="Primário" colorVariant="primary" />
<Button children="Secundário" colorVariant="secondary" />
<Button children="Destaque" colorVariant="accent" />
<Button children="Perigo" colorVariant="danger" />
<Button children="Sucesso" colorVariant="success" />
<Button children="Alerta" colorVariant="warning" />
<Button children="Informação" colorVariant="info" />
```

## Ícones

Você pode adicionar ícones à esquerda ou à direita do texto:

```tsx
// Ícone à esquerda
<Button
  children="Adicionar Usuário"
  leftIconName="user-plus"
  colorVariant="primary"
/>

// Ícone à direita
<Button
  children="Próximo"
  rightIconName="arrow-right"
  colorVariant="primary"
/>

// Ambos os ícones
<Button
  children="Compartilhar"
  leftIconName="share"
  rightIconName="arrow-right"
  colorVariant="primary"
/>
```

## Estados

### Desabilitado

```tsx
<Button children="Botão Desabilitado" disabled colorVariant="primary" />
```

### Carregamento

```tsx
<Button children="Carregando..." loading colorVariant="primary" />
```

## Tamanhos e Formato

Você pode controlar o tamanho e o formato do botão:

```tsx
// Tamanho específico
<Button
  children="Botão Pequeno"
  minWidth="100px"
  colorVariant="primary"
/>

<Button
  children="Botão Grande"
  minWidth="250px"
  maxWidth="300px"
  colorVariant="primary"
/>

// Botão arredondado
<Button
  children="Botão Arredondado"
  rounded
  colorVariant="primary"
/>
```

## Acessibilidade

O componente Button implementa as melhores práticas de acessibilidade:

```tsx
<Button
  children="Finalizar Compra"
  accessibility="Finalizar compra"
  accessibilityLabel="Finalizar sua compra e ir para a página de pagamento"
  onPress={handleCheckout}
  colorVariant="primary"
/>
```

## Propriedades

| Propriedade          | Tipo                                                                                 | Padrão    | Descrição                                                |
| -------------------- | ------------------------------------------------------------------------------------ | --------- | -------------------------------------------------------- |
| `children`           | ReactNode                                                                            | -         | Conteúdo do botão                                        |
| `variant`            | 'filled' \| 'ghost' \| 'tint' \| 'outline' \| 'flat'                                 | 'filled'  | Variante visual do botão                                 |
| `colorVariant`       | 'primary' \| 'secondary' \| 'accent' \| 'danger' \| 'success' \| 'warning' \| 'info' | 'primary' | Variante de cor do botão                                 |
| `onPress`            | function                                                                             | -         | Função chamada quando o botão é pressionado              |
| `disabled`           | boolean                                                                              | false     | Define se o botão está desabilitado                      |
| `loading`            | boolean                                                                              | false     | Exibe um indicador de carregamento no lugar do conteúdo  |
| `leftIconName`       | string                                                                               | -         | Nome do ícone à esquerda do texto                        |
| `rightIconName`      | string                                                                               | -         | Nome do ícone à direita do texto                         |
| `leftIconTouchable`  | boolean                                                                              | true      | Define se o ícone esquerdo é sensível ao toque           |
| `rightIconTouchable` | boolean                                                                              | true      | Define se o ícone direito é sensível ao toque            |
| `minWidth`           | string \| number                                                                     | -         | Largura mínima do botão                                  |
| `maxWidth`           | string \| number                                                                     | -         | Largura máxima do botão                                  |
| `rounded`            | boolean                                                                              | false     | Define se o botão terá cantos completamente arredondados |
| `flat`               | boolean                                                                              | false     | Remove a elevação do botão                               |
| `contrast`           | boolean                                                                              | false     | Ajusta cores para melhor contraste                       |
| `typographyVariant`  | string                                                                               | 'md'      | Tamanho do texto do botão                                |
| `style`              | StyleProp\<ViewStyle\>                                                               | [{}]      | Estilos adicionais para o botão                          |
| `textStyle`          | StyleProp\<TextStyle\>                                                               | {}        | Estilos adicionais para o texto do botão                 |
| `accessibility`      | string                                                                               | -         | Rótulo de acessibilidade para leitores de tela           |
| `accessibilityLabel` | string                                                                               | -         | Rótulo de acessibilidade alternativo                     |
| `testID`             | string                                                                               | -         | ID para testes                                           |
| `id`                 | string                                                                               | -         | ID do componente                                         |

## Exemplos

### Botão de Confirmação em Formulário

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Button } from '@platformbuilders/fluid-react-native';

const FormSubmitButton = () => {
  const [isSubmitting, setIsSubmitting] = useState(false);

  const handleSubmit = async () => {
    setIsSubmitting(true);
    try {
      // Simula uma operação assíncrona
      await new Promise((resolve) => setTimeout(resolve, 2000));
      console.log('Formulário enviado com sucesso!');
    } finally {
      setIsSubmitting(false);
    }
  };

  return (
    <View style={{ padding: 16 }}>
      <Button
        children="Enviar Formulário"
        variant="filled"
        colorVariant="primary"
        loading={isSubmitting}
        disabled={isSubmitting}
        onPress={handleSubmit}
        rightIconName={!isSubmitting ? 'check' : undefined}
      />
    </View>
  );
};
```

### Botões de Navegação

```tsx
import React from 'react';
import { View } from 'react-native';
import { Button } from '@platformbuilders/fluid-react-native';
import { useNavigation } from '@react-navigation/native';

const NavigationButtons = () => {
  const navigation = useNavigation();

  return (
    <View style={{ padding: 16, gap: 12 }}>
      <Button
        children="Voltar"
        variant="ghost"
        colorVariant="primary"
        leftIconName="arrow-left"
        onPress={() => navigation.goBack()}
      />

      <Button
        children="Continuar"
        variant="filled"
        colorVariant="primary"
        rightIconName="arrow-right"
        onPress={() => navigation.navigate('NextScreen')}
      />
    </View>
  );
};
```

### Botão de Login com Estado de Carregamento

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Button, TextInput } from '@platformbuilders/fluid-react-native';

const LoginForm = () => {
  const [isLoggingIn, setIsLoggingIn] = useState(false);
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');

  const handleLogin = async () => {
    if (!email || !password) return;

    setIsLoggingIn(true);
    try {
      // Simula uma operação de login
      await new Promise((resolve) => setTimeout(resolve, 2000));
      console.log('Login realizado com sucesso!');
    } finally {
      setIsLoggingIn(false);
    }
  };

  return (
    <View style={{ padding: 16, gap: 16 }}>
      <TextInput
        label="Email"
        value={email}
        onChangeText={setEmail}
        keyboardType="email-address"
        autoCapitalize="none"
      />

      <TextInput
        label="Senha"
        value={password}
        onChangeText={setPassword}
        secureTextEntry
      />

      <Button
        children="Entrar"
        variant="filled"
        colorVariant="primary"
        loading={isLoggingIn}
        disabled={isLoggingIn || !email || !password}
        onPress={handleLogin}
        minWidth="100%"
      />
    </View>
  );
};
```
