# Header

O componente `Header` é utilizado para criar barras de navegação no topo das telas. Ele fornece uma estrutura consistente para exibir título, subtítulo e botões de navegação.

## Importação

```tsx
import { Header } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
// Com título
<Header title="Minha Tela" />

// Com título e subtítulo
<Header
  title="Detalhes do Produto"
  subtitle="Informações adicionais"
/>

// Com conteúdo personalizado
<Header>
  <View style={{ alignItems: 'center' }}>
    <Image
      source={require('../../assets/logo.png')}
      style={{ width: 100, height: 30 }}
    />
  </View>
</Header>
```

## Temas

O Header suporta diferentes temas:

```tsx
// Tema primário (padrão)
<Header
  title="Meu App"
  theme="PRIMARY"
/>

// Tema secundário
<Header
  title="Meu App"
  theme="SECONDARY"
/>
```

## Ícones de Navegação

O Header pode exibir ícones à esquerda e à direita:

```tsx
// Com ícone de volta à esquerda (padrão: arrow-left)
<Header
  title="Detalhes"
  showLeftIcon
  onPressLeftIcon={() => navigation.goBack()}
/>

// Com ícone personalizado à esquerda
<Header
  title="Configurações"
  showLeftIcon
  leftIconName="cog"
  onPressLeftIcon={handleSettings}
/>

// Com ícone à direita
<Header
  title="Mensagens"
  showRightIcon
  rightIconName="bell"
  onPressRightIcon={handleNotifications}
/>

// Com ícones em ambos os lados
<Header
  title="Perfil"
  showLeftIcon
  showRightIcon
  leftIconName="arrow-left"
  rightIconName="edit"
  onPressLeftIcon={() => navigation.goBack()}
  onPressRightIcon={handleEdit}
/>
```

## Acessibilidade

É possível adicionar rótulos de acessibilidade aos ícones:

```tsx
<Header
  title="Pesquisa"
  showLeftIcon
  showRightIcon
  leftIconName="arrow-left"
  rightIconName="search"
  accessibilityLeftIcon="Voltar para a tela anterior"
  accessibilityRightIcon="Pesquisar conteúdo"
  onPressLeftIcon={() => navigation.goBack()}
  onPressRightIcon={handleSearch}
/>
```

## Propriedades

| Propriedade              | Tipo      | Padrão       | Descrição                                        |
| ------------------------ | --------- | ------------ | ------------------------------------------------ |
| `title`                  | string    | -            | Título exibido no header                         |
| `subtitle`               | string    | ''           | Subtítulo exibido abaixo do título               |
| `theme`                  | string    | 'PRIMARY'    | Tema do header ('PRIMARY', 'SECONDARY')          |
| `children`               | ReactNode | -            | Conteúdo personalizado para o header             |
| `showLeftIcon`           | boolean   | true         | Se deve exibir o ícone à esquerda                |
| `showRightIcon`          | boolean   | true         | Se deve exibir o ícone à direita                 |
| `leftIconName`           | string    | 'arrow-left' | Nome do ícone à esquerda                         |
| `rightIconName`          | string    | 'times'      | Nome do ícone à direita                          |
| `onPressLeftIcon`        | function  | -            | Função chamada ao pressionar o ícone à esquerda  |
| `onPressRightIcon`       | function  | -            | Função chamada ao pressionar o ícone à direita   |
| `accessibilityLeftIcon`  | string    | ''           | Rótulo de acessibilidade para o ícone à esquerda |
| `accessibilityRightIcon` | string    | ''           | Rótulo de acessibilidade para o ícone à direita  |

## Exemplos

### Header de Navegação Básica

```tsx
import React from 'react';
import { useNavigation } from '@react-navigation/native';
import { Header } from '@platformbuilders/fluid-react-native';

const MyScreen = () => {
  const navigation = useNavigation();

  return (
    <View style={{ flex: 1 }}>
      <Header
        title="Minha Tela"
        showLeftIcon
        onPressLeftIcon={() => navigation.goBack()}
      />

      {/* Conteúdo da tela */}
    </View>
  );
};
```

### Header com Ações

```tsx
import React from 'react';
import { useNavigation } from '@react-navigation/native';
import { Header } from '@platformbuilders/fluid-react-native';

const ProductDetailsScreen = () => {
  const navigation = useNavigation();

  const handleShare = () => {
    // Lógica para compartilhar o produto
  };

  return (
    <View style={{ flex: 1 }}>
      <Header
        title="Detalhes do Produto"
        subtitle="Smartphone XYZ"
        showLeftIcon
        showRightIcon
        leftIconName="arrow-left"
        rightIconName="share"
        onPressLeftIcon={() => navigation.goBack()}
        onPressRightIcon={handleShare}
        accessibilityLeftIcon="Voltar para a lista de produtos"
        accessibilityRightIcon="Compartilhar este produto"
      />

      {/* Conteúdo da tela */}
    </View>
  );
};
```

### Header com Logo Personalizado

```tsx
import React from 'react';
import { Image, View } from 'react-native';
import { useNavigation } from '@react-navigation/native';
import { Header } from '@platformbuilders/fluid-react-native';

const HomeScreen = () => {
  const navigation = useNavigation();

  const handleProfile = () => {
    navigation.navigate('Profile');
  };

  return (
    <View style={{ flex: 1 }}>
      <Header
        showLeftIcon={false}
        showRightIcon
        rightIconName="user"
        onPressRightIcon={handleProfile}
      >
        <View style={{ alignItems: 'center' }}>
          <Image
            source={require('../../assets/logo.png')}
            style={{ width: 120, height: 40, resizeMode: 'contain' }}
          />
        </View>
      </Header>

      {/* Conteúdo da tela */}
    </View>
  );
};
```
