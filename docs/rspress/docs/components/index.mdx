# Componentes

O Fluid React Native oferece uma ampla variedade de componentes para construir interfaces de usuário móveis consistentes e acessíveis.

## Categorias de Componentes

### Elementos de UI Básicos
- Tipografia
- Botões
- Inputs
- Cards

### Navegação
- TabBar
- BottomSheet
- Drawer

### Formulários
- TextInput
- MaskedTextInput
- Checkbox
- Radio
- DatePicker

### Feedback
- Toast
- Modal
- Alert
- ProgressBar
- ActivityIndicator

### Mídia
- Avatar
- Image
- Icon

## Acessibilidade

Todos os componentes do Fluid React Native foram projetados com acessibilidade em mente, seguindo as diretrizes WCAG 2.1 e as melhores práticas para React Native. 

Para mais informações sobre acessibilidade, consulte a [documentação de acessibilidade](/guide/accessibility).

## Personalização

O Fluid React Native permite personalizar a aparência dos componentes através do sistema de temas. Você pode criar temas personalizados ou estender o tema padrão para atender às necessidades de sua aplicação.

Para mais informações sobre personalização, consulte a [documentação de temas](/guide/theming).
