# Touchable

O componente `Touchable` é um wrapper sobre o `TouchableOpacity` do React Native que adiciona funcionalidades de feedback tátil e melhora a acessibilidade. É utilizado como base para outros componentes clicáveis da biblioteca.

## Importação

```tsx
import { Touchable } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
<Touchable
  accessibility="Botão de exemplo"
  onPress={() => console.log('Pressionado!')}
>
  <Text>Clique aqui</Text>
</Touchable>
```

## Com Feedback Tátil

O componente Touchable suporta feedback tátil através da biblioteca `react-native-haptic-feedback`:

```tsx
<Touchable
  accessibility="Botão com feedback forte"
  haptic="impactHeavy"
  onPress={() => console.log('Feedback forte!')}
>
  <Text>Feedback forte</Text>
</Touchable>
```

## Desabilitado

```tsx
<Touchable
  accessibility="Botão desabilitado"
  disabled={true}
  onPress={() => console.log('Não será executado')}
>
  <Text style={{ opacity: 0.5 }}>Desabilitado</Text>
</Touchable>
```

## Acessibilidade

O componente Touchable implementa automaticamente atributos de acessibilidade:

```tsx
<Touchable
  accessibility="Adicionar ao carrinho"
  accessibilityLabel="Adicionar produto ao carrinho de compras"
  onPress={() => console.log('Adicionado ao carrinho')}
>
  <Icon name="cart" />
  <Text>Adicionar</Text>
</Touchable>
```

## Propriedades

| Propriedade          | Tipo                | Padrão              | Descrição                                        |
| -------------------- | ------------------- | ------------------- | ------------------------------------------------ |
| `accessibility`      | string              | -                   | Descrição de acessibilidade (obrigatório)        |
| `accessibilityLabel` | string              | accessibility       | Rótulo de acessibilidade para leitores de tela   |
| `onPress`            | function(event)     | () => {}            | Função chamada quando o componente é pressionado |
| `haptic`             | HapticFeedbackTypes | 'impactLight'       | Tipo de feedback tátil ao pressionar             |
| `disabled`           | boolean             | false               | Define se o componente está desabilitado         |
| `id`                 | string              | -                   | ID do componente                                 |
| `testID`             | string              | id ou accessibility | ID para testes automatizados                     |
| `children`           | ReactNode           | -                   | Conteúdo a ser renderizado dentro do componente  |

## Exemplos

### Botão Personalizado

```tsx
import React from 'react';
import { Text, View } from 'react-native';
import { Touchable, Icon } from '@platformbuilders/fluid-react-native';

const CustomButton = ({ onPress, label, iconName, disabled }) => (
  <Touchable
    accessibility={label}
    onPress={onPress}
    disabled={disabled}
    haptic="impactMedium"
  >
    <View
      style={{
        flexDirection: 'row',
        alignItems: 'center',
        backgroundColor: disabled ? '#CCCCCC' : '#3366FF',
        paddingVertical: 12,
        paddingHorizontal: 16,
        borderRadius: 8,
      }}
    >
      {iconName && (
        <Icon
          name={iconName}
          size={18}
          color="#FFFFFF"
          style={{ marginRight: 8 }}
        />
      )}
      <Text style={{ color: '#FFFFFF', fontWeight: 'bold' }}>{label}</Text>
    </View>
  </Touchable>
);

const ButtonExample = () => (
  <View style={{ gap: 16, padding: 16 }}>
    <CustomButton
      label="Salvar"
      iconName="save"
      onPress={() => console.log('Salvo!')}
    />

    <CustomButton
      label="Excluir"
      iconName="trash"
      onPress={() => console.log('Excluído!')}
    />

    <CustomButton
      label="Desabilitado"
      iconName="ban"
      disabled={true}
      onPress={() => console.log('Não será chamado')}
    />
  </View>
);
```

### Card Clicável

```tsx
import React from 'react';
import { View, Text, StyleSheet } from 'react-native';
import {
  Touchable,
  Image,
  Typography,
} from '@platformbuilders/fluid-react-native';

const ProductCard = ({ product, onPress }) => (
  <Touchable
    accessibility={`Produto ${product.name}`}
    onPress={() => onPress(product)}
    haptic="impactLight"
  >
    <View style={styles.card}>
      <Image
        source={{ uri: product.image }}
        style={styles.image}
        accessibility={`Imagem do produto ${product.name}`}
      />
      <View style={styles.content}>
        <Typography variant="md" style={styles.title}>
          {product.name}
        </Typography>
        <Typography variant="sm" style={styles.price}>
          R$ {product.price.toFixed(2)}
        </Typography>
      </View>
    </View>
  </Touchable>
);

const styles = StyleSheet.create({
  card: {
    backgroundColor: '#FFFFFF',
    borderRadius: 8,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 2 },
    shadowOpacity: 0.1,
    shadowRadius: 4,
    elevation: 2,
    margin: 8,
    overflow: 'hidden',
  },
  image: {
    width: '100%',
    height: 150,
    resizeMode: 'cover',
  },
  content: {
    padding: 12,
  },
  title: {
    fontWeight: 'bold',
    marginBottom: 4,
  },
  price: {
    color: '#3366FF',
    fontWeight: 'bold',
  },
});

const ProductListExample = () => {
  const products = [
    {
      id: 1,
      name: 'Smartphone XYZ',
      price: 1299.99,
      image: 'https://example.com/phone.jpg',
    },
    {
      id: 2,
      name: 'Laptop ABC',
      price: 3499.99,
      image: 'https://example.com/laptop.jpg',
    },
  ];

  const handlePress = (product) => {
    console.log(`Produto selecionado: ${product.name}`);
  };

  return (
    <View>
      {products.map((product) => (
        <ProductCard key={product.id} product={product} onPress={handlePress} />
      ))}
    </View>
  );
};
```

### Lista de Opções

```tsx
import React from 'react';
import { View } from 'react-native';
import {
  Touchable,
  Icon,
  Typography,
} from '@platformbuilders/fluid-react-native';

const ListItem = ({ title, subtitle, onPress, iconName }) => (
  <Touchable accessibility={title} onPress={onPress}>
    <View
      style={{
        flexDirection: 'row',
        alignItems: 'center',
        padding: 16,
        borderBottomWidth: 1,
        borderBottomColor: '#EEEEEE',
      }}
    >
      <View style={{ flex: 1 }}>
        <Typography variant="md">{title}</Typography>
        {subtitle && (
          <Typography variant="sm" style={{ color: '#666666', marginTop: 4 }}>
            {subtitle}
          </Typography>
        )}
      </View>
      <Icon name={iconName || 'chevron-right'} size={18} color="#999999" />
    </View>
  </Touchable>
);

const SettingsList = () => (
  <View style={{ flex: 1 }}>
    <ListItem
      title="Perfil"
      subtitle="Editar informações pessoais"
      iconName="user"
      onPress={() => console.log('Navegando para Perfil')}
    />

    <ListItem
      title="Notificações"
      subtitle="Gerenciar preferências de notificação"
      iconName="bell"
      onPress={() => console.log('Navegando para Notificações')}
    />

    <ListItem
      title="Segurança"
      subtitle="Atualizar senha e configurações de segurança"
      iconName="lock"
      onPress={() => console.log('Navegando para Segurança')}
    />

    <ListItem
      title="Ajuda"
      subtitle="Perguntas frequentes e suporte"
      iconName="question-circle"
      onPress={() => console.log('Navegando para Ajuda')}
    />
  </View>
);
```
