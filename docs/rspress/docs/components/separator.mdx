# Separator

O componente `Separator` é utilizado para criar divisões visuais entre seções de conteúdo, com a opção de incluir um texto no centro. É comumente usado para separar grupos de itens ou para indicar alternativas, como "ou" entre diferentes métodos de login.

## Importação

```tsx
import { Separator } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
<Separator text="ou" />
```

## Com Margens Personalizadas

Você pode adicionar margens superior e inferior ao separador:

```tsx
<Separator text="ou" marginTop={20} marginBottom={10} />
```

## Exemplos de Uso

### Separador em Formulário de Login

```tsx
import React from 'react';
import { View } from 'react-native';
import {
  Button,
  Separator,
  TextInput,
  Typography,
} from '@platformbuilders/fluid-react-native';

const LoginScreen = () => {
  return (
    <View style={{ padding: 16, gap: 16 }}>
      <Typography
        variant="lg"
        style={{ textAlign: 'center', marginBottom: 16 }}
      >
        Acesse sua conta
      </Typography>

      <TextInput
        label="Email"
        keyboardType="email-address"
        autoCapitalize="none"
      />

      <TextInput label="Senha" secureTextEntry />

      <Button children="Entrar" variant="filled" colorVariant="primary" />

      <Separator text="ou" marginTop={24} marginBottom={24} />

      <Button
        children="Entrar com Google"
        variant="outlined"
        colorVariant="secondary"
      />

      <Button
        children="Entrar com Facebook"
        variant="outlined"
        colorVariant="secondary"
      />
    </View>
  );
};
```

### Separador em Seções de Perfil

```tsx
import React from 'react';
import { View } from 'react-native';
import {
  Icon,
  Separator,
  Typography,
} from '@platformbuilders/fluid-react-native';

const ProfileSection = ({ title, children }) => (
  <View style={{ marginBottom: 16 }}>
    <Typography variant="md" style={{ fontWeight: 'bold', marginBottom: 8 }}>
      {title}
    </Typography>
    {children}
  </View>
);

const ProfileOption = ({ icon, label }) => (
  <View style={{ flexDirection: 'row', alignItems: 'center', padding: 12 }}>
    <Icon name={icon} size={20} style={{ marginRight: 12 }} />
    <Typography variant="sm">{label}</Typography>
  </View>
);

const ProfileScreen = () => {
  return (
    <View style={{ flex: 1, padding: 16 }}>
      <Typography variant="lg" style={{ marginBottom: 24 }}>
        Meu Perfil
      </Typography>

      <ProfileSection title="Conta">
        <ProfileOption icon="user" label="Informações Pessoais" />
        <ProfileOption icon="envelope" label="Email e Notificações" />
        <ProfileOption icon="lock" label="Segurança" />
      </ProfileSection>

      <Separator text="preferências" marginTop={8} marginBottom={8} />

      <ProfileSection title="Configurações">
        <ProfileOption icon="bell" label="Notificações" />
        <ProfileOption icon="moon" label="Tema Escuro" />
        <ProfileOption icon="globe" label="Idioma" />
      </ProfileSection>

      <Separator text="suporte" marginTop={8} marginBottom={8} />

      <ProfileSection title="Ajuda">
        <ProfileOption icon="question-circle" label="FAQ" />
        <ProfileOption icon="headset" label="Suporte" />
        <ProfileOption icon="info-circle" label="Sobre" />
      </ProfileSection>
    </View>
  );
};
```

## Propriedades

| Propriedade    | Tipo   | Padrão | Descrição                            |
| -------------- | ------ | ------ | ------------------------------------ |
| `text`         | string | -      | Texto exibido no centro do separador |
| `marginTop`    | number | 0      | Margem superior em pixels            |
| `marginBottom` | number | 0      | Margem inferior em pixels            |
