# Link

O componente `Link` é utilizado para criar links navegáveis ou textos clicáveis dentro da aplicação. Este componente é construído com base no componente `Touchable` e oferece uma maneira consistente de implementar navegação entre telas ou interações baseadas em texto.

## Importação

```tsx
import { Link } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
<Link onPress={() => console.log('Link pressionado')}>Clique aqui</Link>
```

## Tamanhos

O Link suporta diferentes variantes de tamanho através da propriedade `variant`:

```tsx
<Link variant="min">Link mínimo</Link>
<Link variant="xs">Link extra pequeno</Link>
<Link variant="sm">Link pequeno</Link>
<Link variant="md">Link médio (padrão)</Link>
<Link variant="lg">Link grande</Link>
<Link variant="xl">Link extra grande</Link>
<Link variant="xxl">Link extra extra grande</Link>
<Link variant="max">Link máximo</Link>
```

## Personalização

É possível personalizar o estilo do link:

```tsx
<Link
  style={{
    color: '#FF5500',
    fontWeight: 'bold',
    textDecorationLine: 'underline',
  }}
  onPress={() => navigation.navigate('Details')}
>
  Ver mais detalhes
</Link>
```

## Estado Desabilitado

```tsx
<Link
  disabled={true}
  onPress={() => console.log('Este código não será executado')}
>
  Link desabilitado
</Link>
```

## Acessibilidade

O componente Link inclui recursos de acessibilidade apropriados:

```tsx
<Link
  accessibility="Ir para a página de ajuda"
  accessibilityLabel="Link para a seção de ajuda e suporte"
  onPress={() => navigation.navigate('Help')}
>
  Ajuda e Suporte
</Link>
```

## Propriedades

| Propriedade          | Tipo                   | Padrão | Descrição                                  |
| -------------------- | ---------------------- | ------ | ------------------------------------------ |
| `children`           | ReactNode              | -      | Conteúdo do link (texto)                   |
| `onPress`            | function               | -      | Função chamada quando o link é pressionado |
| `variant`            | string                 | 'md'   | Variante de tamanho do texto               |
| `style`              | StyleProp\<ViewStyle\> | -      | Estilos adicionais para o link             |
| `disabled`           | boolean                | false  | Se o link está desabilitado                |
| `accessibility`      | string                 | -      | Rótulo de acessibilidade                   |
| `accessibilityLabel` | string                 | -      | Rótulo de acessibilidade alternativo       |
| `testID`             | string                 | -      | ID para testes                             |
| `id`                 | string                 | -      | ID do componente                           |

## Exemplos

### Links de Navegação

```tsx
import React from 'react';
import { View } from 'react-native';
import { useNavigation } from '@react-navigation/native';
import { Link, Typography } from '@platformbuilders/fluid-react-native';

const NavigationLinks = () => {
  const navigation = useNavigation();

  return (
    <View style={{ padding: 16 }}>
      <Typography variant="lg" style={{ marginBottom: 16 }}>
        Navegue para:
      </Typography>

      <Link
        variant="lg"
        style={{ color: '#3366FF', marginBottom: 12 }}
        onPress={() => navigation.navigate('Home')}
      >
        Página Inicial
      </Link>

      <Link
        variant="lg"
        style={{ color: '#3366FF', marginBottom: 12 }}
        onPress={() => navigation.navigate('Profile')}
      >
        Meu Perfil
      </Link>

      <Link
        variant="lg"
        style={{ color: '#3366FF', marginBottom: 12 }}
        onPress={() => navigation.navigate('Settings')}
      >
        Configurações
      </Link>
    </View>
  );
};
```

### Links Externos

```tsx
import React from 'react';
import { View, Linking } from 'react-native';
import { Link, Typography } from '@platformbuilders/fluid-react-native';

const ExternalLinks = () => {
  const openURL = (url) => {
    Linking.canOpenURL(url).then((supported) => {
      if (supported) {
        Linking.openURL(url);
      } else {
        console.log(`Não foi possível abrir o URL: ${url}`);
      }
    });
  };

  return (
    <View style={{ padding: 16 }}>
      <Typography variant="lg" style={{ marginBottom: 16 }}>
        Links Úteis:
      </Typography>

      <Link
        variant="md"
        style={{ color: '#3366FF', marginBottom: 12 }}
        onPress={() => openURL('https://reactnative.dev')}
      >
        Documentação do React Native
      </Link>

      <Link
        variant="md"
        style={{ color: '#3366FF', marginBottom: 12 }}
        onPress={() => openURL('https://platformbuilders.io')}
      >
        Website da Platform Builders
      </Link>
    </View>
  );
};
```

### Link com Feedback Visual

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Link, Typography } from '@platformbuilders/fluid-react-native';

const InteractiveLink = () => {
  const [clicked, setClicked] = useState(false);

  const handlePress = () => {
    setClicked(true);
    // Outras ações...
  };

  return (
    <View style={{ padding: 16 }}>
      <Link
        variant="lg"
        style={{
          color: clicked ? '#4CAF50' : '#3366FF',
          fontWeight: clicked ? 'bold' : 'normal',
        }}
        onPress={handlePress}
      >
        {clicked ? '✓ Confirmado' : 'Confirmar'}
      </Link>

      {clicked && (
        <Typography variant="sm" style={{ color: '#4CAF50', marginTop: 8 }}>
          Ação confirmada com sucesso!
        </Typography>
      )}
    </View>
  );
};
```
