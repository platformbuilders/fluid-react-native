# Markdown

O componente `Markdown` é uma forma simples de renderizar conteúdo formatado em Markdown dentro do seu aplicativo React Native. Ele usa a biblioteca `react-native-markdown-display` para transformar texto Markdown em componentes nativos.

## Importação

```tsx
import { Markdown } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
import React from 'react';
import { Markdown } from '@platformbuilders/fluid-react-native';

const MarkdownExample = () => {
  const markdownContent = `
# Título Principal

Este é um parágrafo com **texto em negrito** e *texto em itálico*.

## Subtítulo

### Lista não ordenada:
- Item 1
- Item 2
- Item 3

### Lista ordenada:
1. Primeiro item
2. Segundo item
3. Terceiro item

[Link para documentação](https://www.platformbuilders.com.br)

> Citação de exemplo

\`\`\`
// Bloco de código
const exemplo = "Olá Mundo";
console.log(exemplo);
\`\`\`
  `;

  return <Markdown>{markdownContent}</Markdown>;
};
```

## Com Estilo Personalizado

Você pode personalizar o estilo do texto renderizado:

```tsx
<Markdown
  textStyle={{
    color: '#333333',
    fontSize: 16,
    lineHeight: 24,
    fontFamily: 'Roboto',
  }}
>
  {markdownContent}
</Markdown>
```

## Propriedades

| Propriedade | Tipo      | Padrão | Descrição                                        |
| ----------- | --------- | ------ | ------------------------------------------------ |
| `children`  | string    | -      | Conteúdo Markdown a ser renderizado              |
| `textStyle` | TextStyle | {}     | Estilo de texto aplicado ao conteúdo renderizado |

## Elementos Suportados

O componente Markdown suporta a maioria dos elementos padrão da sintaxe Markdown:

- Cabeçalhos (# h1, ## h2, etc.)
- Textos estilizados (**negrito**, _itálico_, ~~riscado~~)
- Links
- Listas ordenadas e não ordenadas
- Citações
- Blocos de código
- Imagens
- Tabelas
- Linhas horizontais

### Documentação Interativa

```tsx
import React, { useState } from 'react';
import { StyleSheet, View, ScrollView } from 'react-native';
import {
  Markdown,

  TextInput,

Markdown é uma linguagem de marcação leve que você pode usar para adicionar formatação a documentos de texto simples. Criada por John Gruber em 2004, Markdown é hoje uma das linguagens de marcação mais populares do mundo.

## Sintaxe Básica

### Cabeçalhos

# H1
## H2
### H3

### Ênfase

*Itálico* ou _Itálico_
**Negrito** ou __Negrito__
~~Riscado~~

### Listas

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

1. Item 1
2. Item 2
3. Item 3

### Links

[Texto do Link](https://www.exemplo.com)

### Imagens

![Texto Alt](https://via.placeholder.com/150)

### Citações

> Esta é uma citação de exemplo.
> Você pode ter múltiplas linhas.

### Código

\`\`\`javascript
function exemplo() {
  return "Olá Mundo";
}
\`\`\`

\`Código em linha\`

### Tabelas

| Coluna 1 | Coluna 2 | Coluna 3 |
|----------|----------|----------|
| Linha 1  | Dado     | Dado     |
| Linha 2  | Dado     | Dado     |

### Linha Horizontal

---

`);

  return (
    <View style={styles.container}>
      <ScrollView style={styles.preview}>
        <Markdown textStyle={styles.markdownText}>{markdownSource}</Markdown>
      </ScrollView>

      <View style={styles.divider} />
```
