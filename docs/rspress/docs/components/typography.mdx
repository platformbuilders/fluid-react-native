# Typography

O componente `Typography` é utilizado para exibir texto com estilos consistentes em toda a aplicação. Ele segue o sistema tipográfico definido no design system Fluid, garantindo consistência visual.

## Importação

```tsx
import { Typography } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
<Typography>Texto padrão</Typography>
```

## Variantes de Tamanho

O componente suporta diferentes variantes de tamanho para atender a diferentes necessidades de hierarquia textual:

```tsx
<Typography variant="min">Texto mínimo</Typography>
<Typography variant="xxs">Texto extra extra pequeno</Typography>
<Typography variant="xs">Texto extra pequeno</Typography>
<Typography variant="sm">Texto pequeno</Typography>
<Typography variant="md">Texto médio (padrão)</Typography>
<Typography variant="lg">Texto grande</Typography>
<Typography variant="xl">Texto extra grande</Typography>
<Typography variant="xxl">Texto extra extra grande</Typography>
<Typography variant="max">Texto máximo</Typography>
```

## Altura de Linha

É possível controlar a altura de linha do texto:

```tsx
<Typography lineHeightVariant="min">Altura de linha mínima</Typography>
<Typography lineHeightVariant="md">Altura de linha média</Typography>
<Typography lineHeightVariant="max">Altura de linha máxima</Typography>
```

## Texto Animado

O componente suporta também texto animado:

```tsx
<Typography variant="animated">Texto com animação</Typography>
```

## Múltiplas Linhas

Para limitar o número de linhas e truncar o texto com elipses:

```tsx
<Typography numberOfLines={2}>
  Este é um texto longo que será truncado após duas linhas de texto. O texto que
  ultrapassar esse limite será substituído por reticências...
</Typography>
```

## Personalização

Você pode adicionar estilos personalizados:

```tsx
<Typography style={{ color: '#FF5500', fontWeight: 'bold' }}>
  Texto personalizado
</Typography>
```

## Propriedades

| Propriedade         | Tipo                   | Padrão | Descrição                                                                                           |
| ------------------- | ---------------------- | ------ | --------------------------------------------------------------------------------------------------- |
| `variant`           | string                 | 'md'   | Variante de tamanho do texto ('min', 'xxs', 'xs', 'sm', 'md', 'lg', 'xl', 'xxl', 'max', 'animated') |
| `lineHeightVariant` | string                 | 'min'  | Variante de altura de linha ('min', 'xxs', 'xs', 'sm', 'md', 'lg', 'xl', 'xxl', 'max')              |
| `children`          | React.ReactNode        | -      | Conteúdo de texto                                                                                   |
| `numberOfLines`     | number                 | -      | Limita o texto a um número específico de linhas                                                     |
| `style`             | StyleProp\<TextStyle\> | -      | Estilos adicionais                                                                                  |
| `accessibility`     | string                 | -      | Rótulo de acessibilidade                                                                            |
| `id`                | string                 | -      | ID para testes e referência                                                                         |

## Exemplos

### Hierarquia de Títulos

```tsx
<View style={{ gap: 8 }}>
  <Typography variant="xxl" style={{ fontWeight: 'bold' }}>
    Título Principal
  </Typography>

  <Typography variant="xl" style={{ fontWeight: 'bold' }}>
    Subtítulo
  </Typography>

  <Typography variant="lg">
    Introdução ao assunto principal que será discutido nesta seção.
  </Typography>

  <Typography variant="md">
    Parágrafo padrão com informações detalhadas sobre o tópico apresentado. Este
    texto usa o tamanho padrão da tipografia e é adequado para a maioria dos
    conteúdos textuais da aplicação.
  </Typography>

  <Typography variant="sm" style={{ color: '#666666' }}>
    Nota: Informações adicionais em um tamanho menor.
  </Typography>
</View>
```

### Cards de Produto

```tsx
<View
  style={{
    padding: 16,
    borderRadius: 8,
    backgroundColor: '#FFFFFF',
    shadowColor: '#000',
    shadowOpacity: 0.1,
    shadowRadius: 8,
    elevation: 2,
  }}
>
  <Typography variant="lg" style={{ fontWeight: 'bold' }}>
    Smartphone XYZ
  </Typography>

  <Typography variant="md" style={{ marginTop: 8 }}>
    O smartphone mais avançado do mercado com tela AMOLED, câmera de 108MP e
    bateria de longa duração.
  </Typography>

  <Typography
    variant="xl"
    style={{ color: '#4CAF50', fontWeight: 'bold', marginTop: 16 }}
  >
    R$ 2.499,00
  </Typography>

  <Typography variant="sm" style={{ color: '#666666', marginTop: 4 }}>
    Em até 12x sem juros
  </Typography>
</View>
```

### Notificação com Tempo

```tsx
<View
  style={{
    flexDirection: 'row',
    justifyContent: 'space-between',
    padding: 12,
    backgroundColor: '#E3F2FD',
    borderRadius: 6,
  }}
>
  <View style={{ flex: 1 }}>
    <Typography variant="md" style={{ fontWeight: 'bold' }}>
      Nova mensagem recebida
    </Typography>

    <Typography variant="sm" numberOfLines={2} style={{ marginTop: 4 }}>
      João enviou uma solicitação para revisar o documento de projeto.
    </Typography>
  </View>

  <Typography variant="xs" style={{ color: '#666666' }}>
    5m atrás
  </Typography>
</View>
```
