# Toggle

O componente `Toggle` é utilizado para alternar entre dois estados, como ativar ou desativar uma funcionalidade. É similar a um checkbox, mas com uma interface visual diferente que representa melhor o conceito de "ligar/desligar".

## Importação

```tsx
import { Toggle } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
import React, { useState } from 'react';
import { Toggle } from '@platformbuilders/fluid-react-native';

const ExampleToggle = () => {
  const [isEnabled, setIsEnabled] = useState(false);

  return <Toggle value={isEnabled} onValueChange={setIsEnabled} />;
};
```

## Personalização de Cores

Você pode personalizar as cores do botão e da trilha para cada estado (ativado/desativado):

```tsx
<Toggle
  value={isEnabled}
  onValueChange={setIsEnabled}
  thumbColor={{
    trueColor: '#47dd36', // Cor do botão quando ativado
    falseColor: '#df5409', // Cor do botão quando desativado
  }}
  trackColor={{
    trueColor: '#c7e2f8', // Cor da trilha quando ativado
    falseColor: '#c7e2f8', // Cor da trilha quando desativado
  }}
/>
```

## Estado Desabilitado

O toggle pode ser desabilitado, impedindo interação:

```tsx
<Toggle value={isEnabled} onValueChange={setIsEnabled} isDisabled={true} />
```

## Acessibilidade

O componente Toggle implementa recursos de acessibilidade apropriados:

```tsx
<Toggle
  value={isDarkMode}
  onValueChange={toggleDarkMode}
  accessibility="Modo escuro"
  // O componente automaticamente configura:
  // - accessibilityRole="switch"
  // - accessibilityState={{ checked: value, disabled: isDisabled }}
  // - accessibilityHint="Double tap to toggle the switch"
/>
```

## Propriedades

| Propriedade             | Tipo     | Padrão                               | Descrição                                                              |
| ----------------------- | -------- | ------------------------------------ | ---------------------------------------------------------------------- |
| `value`                 | boolean  | -                                    | Estado atual do toggle (true = ativado, false = desativado)            |
| `onValueChange`         | function | -                                    | Função chamada quando o valor muda, recebe o novo valor como parâmetro |
| `isDisabled`            | boolean  | false                                | Define se o toggle está desabilitado                                   |
| `thumbColor`            | object   | -                                    | Define as cores do botão do toggle nos estados ativado/desativado      |
| `thumbColor.trueColor`  | string   | Cor primária do tema                 | Cor do botão quando o toggle está ativado                              |
| `thumbColor.falseColor` | string   | backgroundZ4 do tema                 | Cor do botão quando o toggle está desativado                           |
| `trackColor`            | object   | -                                    | Define as cores da trilha do toggle nos estados ativado/desativado     |
| `trackColor.trueColor`  | string   | Cor primária do tema com 30% opacity | Cor da trilha quando o toggle está ativado                             |
| `trackColor.falseColor` | string   | backgroundZ3 do tema                 | Cor da trilha quando o toggle está desativado                          |
| `accessibility`         | string   | -                                    | Texto de acessibilidade para leitores de tela                          |
| `id`                    | string   | -                                    | ID do componente, usado para testes e referências                      |

## Exemplos

### Toggle de Modo Escuro

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Toggle, Typography } from '@platformbuilders/fluid-react-native';

const DarkModeToggle = () => {
  const [darkMode, setDarkMode] = useState(false);

  return (
    <View style={{ flexDirection: 'row', alignItems: 'center', padding: 16 }}>
      <Typography variant="md" style={{ flex: 1 }}>
        Modo Escuro
      </Typography>
      <Toggle
        value={darkMode}
        onValueChange={setDarkMode}
        accessibility="Alternar modo escuro"
        thumbColor={{
          trueColor: '#333333',
          falseColor: '#FFFFFF',
        }}
        trackColor={{
          trueColor: '#666666',
          falseColor: '#DDDDDD',
        }}
      />
    </View>
  );
};
```

### Configurações de Notificações

```tsx
    <View style={{ padding: 16 }}>
      <Typography variant="lg" style={{ marginBottom: 16 }}>
        Configurações de Notificações
      </Typography>

      <View style={{ gap: 16 }}>
        <View style={{ flexDirection: 'row', alignItems: 'center' }}>
          <Typography variant="md" style={{ flex: 1 }}>
            Notificações Push
          </Typography>
          <Toggle
            value={pushEnabled}
            onValueChange={setPushEnabled}
            accessibility="Ativar notificações push"
          />
        </View>

        <View style={{ flexDirection: 'row', alignItems: 'center' }}>
          <Typography variant="md" style={{ flex: 1 }}>
            Notificações por Email
          </Typography>
          <Toggle
            value={emailEnabled}
            onValueChange={setEmailEnabled}
            accessibility="Ativar notificações por email"
          />
        </View>

        <View style={{ flexDirection: 'row', alignItems: 'center' }}>
          <Typography variant="md" style={{ flex: 1 }}>
            Notificações por SMS
          </Typography>
          <Toggle
            value={smsEnabled}
            onValueChange={setSmsEnabled}
            accessibility="Ativar notificações por SMS"
          />
        </View>
      </View>
    </View>
  );
};
```
