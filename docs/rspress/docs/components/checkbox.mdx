# Checkbox

O componente `Checkbox` é utilizado para permitir que o usuário selecione uma ou mais opções em um conjunto. É frequentemente usado em formulários, configurações e listas de seleção múltipla.

## Importação

```tsx
import { Checkbox } from '@platformbuilders/fluid-react-native';
```

## Uso Básico

```tsx
import React, { useState } from 'react';
import { Checkbox } from '@platformbuilders/fluid-react-native';

const ExampleCheckbox = () => {
  const [isChecked, setIsChecked] = useState(false);

  return (
    <Checkbox
      label="Aceito os termos de uso"
      checked={isChecked}
      onPress={() => setIsChecked(!isChecked)}
    />
  );
};
```

## Com Mensagem de Erro

O componente pode exibir uma mensagem de erro abaixo dele:

```tsx
import React, { useState } from 'react';
import { Checkbox } from '@platformbuilders/fluid-react-native';

const ValidationCheckbox = () => {
  const [isChecked, setIsChecked] = useState(false);
  const [submitted, setSubmitted] = useState(false);

  const handleSubmit = () => {
    setSubmitted(true);
    // Outros processos de submissão...
  };

  return (
    <Checkbox
      label="Aceito os termos de uso"
      checked={isChecked}
      onPress={() => setIsChecked(!isChecked)}
      error={
        submitted && !isChecked
          ? 'Você precisa aceitar os termos para continuar'
          : ''
      }
    />
  );
};
```

## Estilo Personalizado

É possível personalizar o estilo do label:

```tsx
<Checkbox
  label="Opção destacada"
  checked={isChecked}
  onPress={() => setIsChecked(!isChecked)}
  labelStyle={{ color: '#3366FF', fontWeight: 'bold' }}
/>
```

## Em Formulários

O checkbox é comumente usado em conjunto com outros componentes de formulário:

```tsx
<View>
  <Typography variant="title">Preferências</Typography>

  <Checkbox
    label="Receber notificações"
    checked={notifications}
    onPress={() => setNotifications(!notifications)}
  />

  <Checkbox
    label="Aceitar emails promocionais"
    checked={promotionalEmails}
    onPress={() => setPromotionalEmails(!promotionalEmails)}
  />
</View>
```

## Propriedades

| Propriedade     | Tipo                   | Padrão   | Descrição                                      |
| --------------- | ---------------------- | -------- | ---------------------------------------------- |
| `label`         | string                 | ''       | Texto exibido ao lado do checkbox              |
| `checked`       | boolean                | false    | Estado de seleção do checkbox                  |
| `onPress`       | function               | () => {} | Função chamada quando o checkbox é pressionado |
| `error`         | string                 | ''       | Mensagem de erro a ser exibida                 |
| `labelStyle`    | StyleProp\<TextStyle\> | -        | Estilo do texto do label                       |
| `accessibility` | string                 | -        | Rótulo de acessibilidade                       |
| `id`            | string                 | -        | ID do componente                               |

## Exemplos

### Lista de Opções com Checkbox

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Checkbox, Typography } from '@platformbuilders/fluid-react-native';

const CheckboxList = () => {
  const options = [
    { id: '1', label: 'Opção 1' },
    { id: '2', label: 'Opção 2' },
    { id: '3', label: 'Opção 3' },
  ];

  const [selectedOptions, setSelectedOptions] = useState([]);

  const toggleOption = (optionId) => {
    if (selectedOptions.includes(optionId)) {
      setSelectedOptions(selectedOptions.filter((id) => id !== optionId));
    } else {
      setSelectedOptions([...selectedOptions, optionId]);
    }
  };

  return (
    <View style={{ padding: 16 }}>
      <Typography variant="lg" style={{ marginBottom: 16 }}>
        Selecione as opções desejadas:
      </Typography>

      <View style={{ gap: 12 }}>
        {options.map((option) => (
          <Checkbox
            key={option.id}
            label={option.label}
            checked={selectedOptions.includes(option.id)}
            onPress={() => toggleOption(option.id)}
          />
        ))}
      </View>

      <Typography variant="sm" style={{ marginTop: 16, color: '#666666' }}>
        Opções selecionadas: {selectedOptions.length}
      </Typography>
    </View>
  );
};
```

### Formulário com Validação

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import {
  Button,
  Checkbox,
  TextInput,
  Typography,
} from '@platformbuilders/fluid-react-native';

const SubscriptionForm = () => {
  const [name, setName] = useState('');
  const [email, setEmail] = useState('');
  const [receiveNewsletter, setReceiveNewsletter] = useState(false);
  const [acceptTerms, setAcceptTerms] = useState(false);
  const [submitted, setSubmitted] = useState(false);

  const handleSubmit = () => {
    setSubmitted(true);

    if (name && email && acceptTerms) {
      console.log('Formulário enviado:', { name, email, receiveNewsletter });
      // Enviar dados para o servidor
    }
  };

  return (
    <View style={{ padding: 16, gap: 16 }}>
      <Typography variant="lg">Inscrição na Newsletter</Typography>

      <TextInput
        label="Nome"
        value={name}
        onChangeText={setName}
        error={submitted && !name ? 'Nome é obrigatório' : ''}
      />

      <TextInput
        label="Email"
        value={email}
        onChangeText={setEmail}
        keyboardType="email-address"
        error={submitted && !email ? 'Email é obrigatório' : ''}
      />

      <Checkbox
        label="Desejo receber a newsletter semanal"
        checked={receiveNewsletter}
        onPress={() => setReceiveNewsletter(!receiveNewsletter)}
      />

      <Checkbox
        label="Aceito os termos de uso e política de privacidade"
        checked={acceptTerms}
        onPress={() => setAcceptTerms(!acceptTerms)}
        error={
          submitted && !acceptTerms
            ? 'Você precisa aceitar os termos para continuar'
            : ''
        }
      />

      <Button
        children="Inscrever-se"
        variant="filled"
        colorVariant="primary"
        onPress={handleSubmit}
      />
    </View>
  );
};
```

### Configurações de Preferências

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Checkbox, Typography } from '@platformbuilders/fluid-react-native';

const PreferencesSettings = () => {
  const [preferences, setPreferences] = useState({
    notifications: true,
    darkMode: false,
    autoUpdate: true,
    dataSaving: false,
    emailAlerts: true,
  });

  const togglePreference = (key) => {
    setPreferences({
      ...preferences,
      [key]: !preferences[key],
    });
  };

  return (
    <View style={{ padding: 16 }}>
      <Typography variant="lg" style={{ marginBottom: 16 }}>
        Configurações e Preferências
      </Typography>

      <View style={{ gap: 16 }}>
        <View>
          <Typography
            variant="md"
            style={{ fontWeight: 'bold', marginBottom: 8 }}
          >
            Notificações
          </Typography>
          <Checkbox
            label="Permitir notificações push"
            checked={preferences.notifications}
            onPress={() => togglePreference('notifications')}
          />
          <Checkbox
            label="Receber alertas por email"
            checked={preferences.emailAlerts}
            onPress={() => togglePreference('emailAlerts')}
          />
        </View>

        <View>
          <Typography
            variant="md"
            style={{ fontWeight: 'bold', marginBottom: 8 }}
          >
            Aparência
          </Typography>
          <Checkbox
            label="Modo escuro"
            checked={preferences.darkMode}
            onPress={() => togglePreference('darkMode')}
          />
        </View>

        <View>
          <Typography
            variant="md"
            style={{ fontWeight: 'bold', marginBottom: 8 }}
          >
            Dados e Atualizações
          </Typography>
          <Checkbox
            label="Atualizar automaticamente"
            checked={preferences.autoUpdate}
            onPress={() => togglePreference('autoUpdate')}
          />
          <Checkbox
            label="Modo de economia de dados"
            checked={preferences.dataSaving}
            onPress={() => togglePreference('dataSaving')}
          />
        </View>
      </View>
    </View>
  );
};
```
