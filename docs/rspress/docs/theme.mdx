# Sistema de Temas

O Fluid React Native inclui um poderoso sistema de temas que permite estilizar de maneira consistente toda a sua aplicação. O sistema é baseado em tokens de design que definem cores, tipografia, espaçamento e outros estilos visuais.

## Conceitos Básicos

O sistema de temas do Fluid é organizado em tokens de design que são utilizados como base para todos os componentes. Os tokens são agrupados em categorias, como cores, espaçamento, tipografia, etc.

## Tokens Disponíveis

O tema padrão inclui tokens para:

- **Cores** - Paleta de cores da marca, tons de cinza, cores de estado
- **Tipografia** - Famílias de fonte, tamanhos, pesos, estilos
- **Espaçamento** - Sistema de espaçamento consistente
- **Bordas** - Raios de borda, espessuras
- **Sombras** - Elevações e sombras
- **Tamanhos** - Sistema de tamanhos para componentes
- **Animações** - Durações e easing para animações

## Utilizando o Tema

O sistema de temas é acessível em todos os componentes estilizados através do `styled-components`. Você pode acessar valores do tema usando a função `getTheme`:

```tsx
import styled from 'styled-components/native';
import { getTheme } from '@platformbuilders/fluid-react-native';

const Container = styled.View`
  padding: ${getTheme('spacing.md')};
  background-color: ${getTheme('colors.background')};
  border-radius: ${getTheme('borderRadius.sm')};
`;
```

## Temas Claro e Escuro

O Fluid suporta temas claro e escuro por padrão. Você pode alternar entre eles conforme necessário:

```tsx
import React, { useState } from 'react';
import { ThemeProvider } from 'styled-components/native';
import { lightTheme, darkTheme } from '@platformbuilders/fluid-react-native';

const App = () => {
  const [isDarkMode, setIsDarkMode] = useState(false);

  return (
    <ThemeProvider theme={isDarkMode ? darkTheme : lightTheme}>
      {/* Sua aplicação aqui */}
      <Button
        title="Alternar Tema"
        onPress={() => setIsDarkMode(!isDarkMode)}
      />
    </ThemeProvider>
  );
};
```

## Personalizando o Tema

Você pode personalizar o tema para atender às necessidades do seu projeto:

```tsx
import { ThemeProvider } from 'styled-components/native';
import { theme as fluidTheme } from '@platformbuilders/fluid-react-native';

// Personalização do tema
const customTheme = {
  ...fluidTheme,
  brand: {
    ...fluidTheme.brand,
    primary: {
      main: '#0066CC',
      contrast: '#FFFFFF',
    },
  },
  spacing: {
    ...fluidTheme.spacing,
    md: '16px',
  },
  typography: {
    ...fluidTheme.typography,
    fontFamily: {
      regular: 'CustomFont-Regular',
      medium: 'CustomFont-Medium',
      bold: 'CustomFont-Bold',
    },
  },
};

const App = () => (
  <ThemeProvider theme={customTheme}>{/* Sua aplicação aqui */}</ThemeProvider>
);
```

## Composição de Temas

Para projetos maiores, você pode dividir a personalização do tema em módulos:

```tsx
import { theme as fluidTheme } from '@platformbuilders/fluid-react-native';

// Módulos de tema
const brandColors = {
  primary: {
    main: '#0066CC',
    contrast: '#FFFFFF',
  },
  secondary: {
    main: '#FF6B00',
    contrast: '#FFFFFF',
  },
};

const typography = {
  fontFamily: {
    regular: 'CustomFont-Regular',
    medium: 'CustomFont-Medium',
    bold: 'CustomFont-Bold',
  },
};

// Tema composto
const customTheme = {
  ...fluidTheme,
  brand: {
    ...fluidTheme.brand,
    ...brandColors,
  },
  typography: {
    ...fluidTheme.typography,
    ...typography,
  },
};
```
