# Fluid React Native

Bem-vindo(a) à documentação oficial do **Fluid React Native**, a implementação do design system Fluid para React Native, desenvolvida pela Platform Builders.

## Visão Geral

O Fluid React Native é uma biblioteca de componentes que implementa o design system Fluid em aplicações React Native. Ele oferece um conjunto completo de componentes reutilizáveis, prontos para uso, seguindo as melhores práticas de UI/UX e acessibilidade.

## Componentes Documentados

A documentação dos componentes está em andamento. Atualmente, temos documentação detalhada para:

- [Avatar](./components/avatar) - Para exibição de imagens de perfil e monogramas
- [Button](./components/button) - Para ações e interações principais
- [Checkbox](./components/checkbox) - Para seleção de múltiplas opções
- [TextInput](./components/textinput) - Para entrada de texto em formulários

A lista completa de componentes disponíveis pode ser encontrada na [página de componentes](./components).

## Desenvolvimento com Storybook

O projeto utiliza o [Storybook](./storybook) para o desenvolvimento e teste de componentes isolados. Isso permite:

- Visualizar componentes em diferentes estados
- Testar interativamente diversas configurações
- Documentar o uso e comportamento dos componentes
