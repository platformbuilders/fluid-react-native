# Fluid React Native

Bem-vindo à documentação do Fluid React Native - uma biblioteca de componentes para React Native que implementa o Design System Fluid da Platform Builders.

## O que é o Fluid React Native?

O Fluid React Native é uma biblioteca completa de componentes reutilizáveis, projetados para facilitar o desenvolvimento de aplicações móveis com React Native. Oferecemos componentes prontos para uso, altamente personalizáveis e acessíveis.

## Por que usar o Fluid React Native?

- **Desenvolvimento rápido**: componentes prontos para uso
- **Consistência visual**: implementação do Design System Fluid
- **Acessibilidade**: componentes projetados pensando em acessibilidade
- **Testabilidade**: alta cobertura de testes e padrões de testID
- **Personalização**: temas ajustáveis para suas necessidades
